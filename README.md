## About Yadavar

Yadavar helps users manage events, reminders, anniversaries, birthdays, and Persian calendar dates. The app is designed with local-first privacy:

- Events and reminders are stored on your device.
- Local backup/export is user controlled.
- Google Drive backup is optional and uses the restricted `drive.appdata` app folder.
- Backup encryption is optional and uses AES-256-GCM.
- Reminder notifications are local device notifications.
- Public Persian holiday data can be fetched from third-party calendar APIs.
- The app does not use ads or third-party tracking SDKs.
