# Changelog

Notable public-support and app-documentation updates for Yadavar.

## 2.3.0 - July 2026

### Added

- Public support repository for privacy information, support questions, bug reports, and feature requests.
- Official Privacy Policy and Support URLs for App Store metadata.
- Optional Google Drive backup documentation for the restricted `drive.appdata` app folder.
- Optional AES-256-GCM backup encryption documentation.
- Public issue templates for bugs, feature requests, and support questions.

### App Notes

- Events, reminders, categories, and app settings are stored locally on device.
- Local backup/export and Google Drive backup are user-initiated.
- Reminder notifications are local device notifications.
- Persian holiday data can be fetched from public third-party calendar APIs.
